# afficheur-3d
petite application qui affiche des modèles 3D avec Vulkan écrite en Zig
cette application n'a seulement étée testé sur Linux
# dépendances Fedora
glfw-devel vulkan-headers vulkan-loader-devel vulkan-utility-libraries-devel glslc
# Installer Zig 0.16.x
https://ziglang.org/download/
après avoir installé zig, il suffit d'utiliser la commande `zig build` dans la repo afin de compiler
